# CSGO Dynamic Text Test
Test new entity "point_worldtext" which is added in CSGO Hydra Opreation update.  
This plugin spawns an entity "point_worldtext" at where you aim.

## Usage
sm_dtt "yourtext" "yourtextsize" "r" "g" "b"
